# 🎯 Guess the Number - GUI Edition

A fun and interactive number guessing game with a clean graphical interface built with Python and tkinter.

## 🎮 How to Play

1. The computer picks a random number between 1 and 100
2. Type your guess in the text box
3. Click "Send" or press Enter
4. Get instant feedback:
- ⬆️ Too high
- ⬇️ Too low
- 🎉 Correct!
5. Try to guess the number in as few attempts as possible!

## ✨ Features

- 🖥️ Clean and modern GUI with tkinter
- 🔢 Random number generation (1-100)
- 📊 Instant feedback with colors (orange for hints, green for success, red for errors)
- 🔄 Replay option without closing the app
- 🛡️ Input validation (handles non-numeric entries and out-of-range numbers)
- 🎨 Color-coded messages for better user experience

## 🚀 How to Run

1. Make sure you have Python installed (version 3.6+)
2. Download or clone this repository
3. Run the program:

```bash
python guess_number_tkinter.py
```
